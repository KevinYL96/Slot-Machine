# Slot-Machine

**Description**
---
Python based slot-machine game.

**Getting Started**
---
**Dependencies**
1. VSCode
2. Python
3. Python Interpreter

**Installing**
1. https://code.visualstudio.com/download
2. https://www.python.org/

**Executing Program**
---
1. Press ENTER to play.
2. Press Q + ENTER to quit the game.
